- نوع المشروع (موقع تعريفي / متجر / حجوزات / لوحة تحكم / ...)
- اللغات المدعومة
- الجمهور المستهدف

### 2. البنية التقنية
- Node.js version
- Framework (Express / Fastify / Next.js / ...)
- Frontend (React / Vue / Vanilla / EJS / ...)
- Database (PostgreSQL / SQLite / MongoDB / ...)
- ORM (Drizzle / Prisma / Sequelize / None)
- Package manager (npm / pnpm / yarn)
- Authentication system
- File storage
- مكتبات رئيسية في package.json

### 3. هيكل المشروع
