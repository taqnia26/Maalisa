Build a complete, production-grade bilingual hotel website for "نزل المعالي / Nuzul Al-Ma'ali" with a full booking system and dynamic admin dashboard.

---

## TECH STACK
- Frontend: HTML5, CSS3, Vanilla JavaScript (or React if preferred)
- Backend: Node.js + Express
- Database: SQLite (via better-sqlite3) or JSON file-based for simplicity
- No external CSS frameworks — write custom CSS only
- Fonts: Google Fonts — "Tajawal" for Arabic, "Cormorant Garamond" for English headings, "Lato" for body text

- Primary Gold: #C9A84C

## BILINGUAL SUPPORT (AR/EN)

- Full RTL for Arabic, LTR for English
- Language toggle button fixed in the navbar (AR | EN)
- All text stored in a `translations.js` object — switching language updates ALL text on the page without reload
- Default language: Arabic
- When Arabic is active: `dir="rtl"` on `<html>`, font switches to Tajawal
- When English is active: `dir="ltr"`, font switches to Cormorant/Lato

## BOOKING SYSTEM

### Guest Booking Flow:
1. Select check-in / check-out dates (date picker)
2. Select room type from dropdown
3. Select number of guests
4. Choose: "Book as Guest" OR "Login / Register"
5. Guest booking: name, phone, email (no account required)
6. Registered user: login → booking pre-fills profile data
7. Booking summary page → Confirm button
8. Confirmation page with booking reference number

### Booking Rules:
- Prevent double-booking (check availability against existing bookings)
- Minimum stay: 1 night
- Date validation: check-out must be after check-in
